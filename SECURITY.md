# Security Policy

## Reporting a Vulnerability
If you discover a security vulnerability within Washtech Direct repository, please send an email to security[@]washtechdirect.com with a detailed description of the issue. 

**Do not report security vulnerabilities through public GitHub issues.**

We take security seriously, and all valid reports will be promptly addressed.

## Preferred Languages

We prefer all communications to be in English.

## Policy

Washtech Direct follows the principle of [Coordinated Vulnerability Disclosure](https://en.wikipedia.org/wiki/Coordinated_vulnerability_disclosure).
